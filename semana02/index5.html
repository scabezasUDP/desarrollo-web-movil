Ejercicio

Creación de una Web Personal Básica

Objetivo

Crear una página web simple y personal que contenga información sobre el estudiante.

Descripción del ejercicio

Los estudiantes deberán crear un único archivo index.html. El objetivo es que apliquen los conocimientos de las primeras clases sobre estructura, etiquetas de texto, imágenes y enlaces.

Instrucciones paso a paso

  1. Estructura básica
  2. Encabezado y foto
  3. Sección de pasatiempos
  4. Sección de contacto y enlaces
